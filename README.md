# Airbnb_Performance_Dashboard

🌍 Global Airbnb Performance Dashboard
- Airbnb analytics dashboard showcasing trends, performance metrics, and customer behavior using Power BI.

📖 Project Overview
- This project presents an interactive Power BI dashboard built to analyze global Airbnb data and uncover meaningful insights related to revenue, customer behavior, and listing performance.
The dashboard enables users to explore patterns, compare regions, and make data-driven decisions through visually rich and dynamic reports.

🎯 Objectives
- Analyze Airbnb listings across different locations
- Identify top-performing cities and hosts
- Understand customer review behavior
- Track revenue trends and booking patterns
- Provide actionable insights for business growth

📊 Key Insights
- 🌍 New York, London, and Paris generate the highest revenue, indicating strong demand in major tourist cities
- 💰 Hong Kong contributes significantly less compared to other destinations, as reflected in lower ratings for parameters such as cleanliness and value for money
- 📊 Revenue distribution is uneven, with the top 4 cities contributing to 58% of the total revenue
- 📅 Booking activity shows seasonal trends, with peak demand during holidays and vacation periods

📊 Business Insights
- 🎯 Hosts should focus on maintaining high ratings and competitive pricing to maximize occupancy
- 🏆 Investing in top-performing locations yields higher returns compared to less popular regions

🛠️ Tech Stack
- Power BI Desktop: Main data visualization platform used for report creation.
- Power Query: Data transformation and cleaning layer for reshaping and preparing the data.
- DAX (Data Analysis Expressions): Used for calculated measures, dynamic visuals, and conditional logic.
- File Formats: .pbix for development and .png for dashboard previews.

📊 Walkthrough of Key Visuals
- 📌 KPI Cards (Top Section)
Why used: KPI cards provide a quick snapshot of overall performance, allowing users to instantly understand key metrics without exploring the entire dashboard.

- 📍 Revenue by Location (Bar Chart)
Why used: Helps identify top-performing locations and compare geographical performance, making it easier to spot high-demand areas.

- 📈 Growth of the Company Over the Years (Line Chart)
Why used: Line charts are ideal for time-series analysis and help in identifying trends and growth patterns over time.

- ⭐ Ratings Across Cities (Matrix Chart)
Why used: Used to analyze how customer satisfaction impacts engagement and booking behavior through multi-dimensional comparison.

- 📊 Seasonality Analysis (Ribbon Chart)
Why used: A ribbon chart is ideal for showing rank changes over time. It helps track how different cities move up or down in terms of performance across months, making it easy to identify seasonal trends and shifting dominance.

📁 Data Source
The dataset used in this project is sourced from Maven Analytics, a well-known platform for data analytics learning and real-world datasets.

📌 Dataset Features:
- 📍 Location details (city, neighborhood)
- 💰 Pricing and availability information
- ⭐ Customer ratings and reviews
- 👤 Host and reviewer data

The dataset was cleaned and transformed using Power Query in Power BI to handle missing values and ensure data consistency before analysis.
